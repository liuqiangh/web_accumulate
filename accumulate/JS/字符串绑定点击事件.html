# html字符串绑定点击事件

像htmls字符串可以加很多事件了，可以以变量的形式，添加dom，function等等，知道一下这个东西，以后说不定用得着。

// html 部分
<div id="aa"></div>
// js部分
function test() {
  console.log('执行函数');
}
const htmls = `<div onclick="test()">插入的html</div>`; // 可以以变量的形式添加dom、函数等等
let a = document.querySelector('#aa');
a.innerHTML = htmls;
